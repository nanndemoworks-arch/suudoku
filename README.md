# Sudoku Trainer

Reactとバックトラック法（アルゴリズム）を用いた、無限レベル生成機能を備えた本格数独アプリです。

## 特徴
- **無限レベル生成**: 難易度を数値で指定し、その場で新しい盤面をアルゴリズム生成します。
- **トレーニング機能**: 候補オート入力、ヒント表示、誤りチェックなど、脳トレに最適な機能を搭載。
- **レスポンシブデザイン**: Tailwind CSSによるモバイル対応のクリーンなUI。
- **データ保存**: ローカルストレージを利用した中断・再開機能。

## 技術スタック
- **Framework**: Next.js (App Router)
- **Library**: React
- **Styling**: Tailwind CSS
- **Algorithm**: Backtracking Algorithm (MRV Heuristic)
